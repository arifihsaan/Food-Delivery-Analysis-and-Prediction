# 🍔 Food Delivery Analysis & Delivery Time Prediction

## 📌 Project Overview
This project focuses on analyzing food delivery data and building a machine learning model to predict delivery time. The goal is to understand key factors that influence delivery duration and provide insights that can help improve operational efficiency.

Through this project, both data analysis and predictive modeling techniques are applied to extract meaningful insights and build an accurate prediction system.

---

## 🎯 Objectives
- Analyze factors affecting delivery time
- Build a machine learning model to predict delivery duration
- Provide insights to improve delivery performance
- Develop an interactive application for real-time prediction

---

## 📊 Dataset
The dataset contains information related to food delivery operations, including:
- Delivery distance
- Preparation time
- Weather conditions
- Traffic conditions
- Vehicle type
- Order details
